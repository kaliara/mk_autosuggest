Overview

Custom written autosuggestion jQuery plugin includes the following features:

  • remote source with parameters
  • efficient, in-memory rendering of results
  • multiple result sets
  • multiple search term highlighting
  • aborting unfinished requests
  • simple customization of item rendering for adding attributes, etc
  • keyboard navigation (IE7+, FF, Chrome)


Demo

Try typing "pizza" to get a sense of the interaction. Each keystroke sends another request to the location specified by the "url" parameter. This demo uses a static JSON file, so the results will never change.

Usage

$("#query").mkAutosuggest({ 
  url: '/api/autosuggest', 
  next_input_element: "#search",
  result_groups: ["categories", "businesses"],
  result_group_labels: ["categories", "businesses"]
});


Requirements

  • jQuery 1.7.2+
  • properly formatted JSON results
